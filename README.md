# Курсовая работа по алгоритмам и структурам данных

Информационная система для обслуживания читателей в библиотеке.

## Описание

Проект представляет собой информационную систему для управления библиотекой, которая осуществляет ввод, хранение, обработку и вывод данных о:

- Читателях библиотеки
- Книгах в фонде библиотеки
- Выдаче и приеме книг

## Технологии

- **Язык программирования:** Kotlin
- **Платформы:** JVM, WASM (Kotlin Multiplatform)
- **UI:** Compose Multiplatform
- **Система сборки:** Gradle

## Структура проекта

```
course-algs/
├── core/           # Основные модели и структуры данных
├── app/            # UI приложение (Compose Multiplatform)
├── docs/           # Документация (Typst)
└── VARIANT.md      # Описание варианта задания
```

### Модули

#### `core`
Содержит реализацию основных структур данных и бизнес-логики:

- **Хеш-таблица** (открытое хеширование) — для хранения данных о читателях
- **АВЛ-дерево** — для хранения данных о книгах
- **Слоеный список** — для учета выдачи и приема книг
- **Алгоритмы:**
  - Сортировка включением
  - Симметричный обход дерева
  - Поиск подстроки (алгоритм Боера-Мура)

#### `app`
Пользовательский интерфейс на базе Compose Multiplatform для взаимодействия с системой.

#### `docs`
Документация проекта в формате Typst.

## Сборка и запуск

### Требования
- JDK с 11 до 21

### Команды

```bash
# Сборка проекта
./gradlew build

# Запуск приложения
./gradlew :app:run

# Генерация документации
./gradlew :docs:buildTypst
```
